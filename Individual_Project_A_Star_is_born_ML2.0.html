Predicting the Star Ratings of Amazon Reviews using Neural Networks

1. Executive Summary

Text Classification is of prime importance in Machine Learning. One of the major concens using text Data for classification is that it often involves operating with highly sparse data. What makes these problems unique and somewhat difficult to handle? In ML 1.0 we found out that Logistic Regression was the best method to do text classification and we achieved a test accuracy of around 72%. The simplest neural networks - the Multilayer Perceptrons (MLP) have the same objective function as the Logistic Regression. Does that mean that MLP's inherently work well for text classification? This project helps to unearth some aspects regarding the same. At the end, we would have a clear cut mental model as to how to tackle sparse data more so text classification problems in Machine Learning.

Problem Statement:

" Identify the viability of Multi-layer Perceptrons in handling problems related to text classification" </font>

Data Description:

Over 1000 reviews of Amazon products with their corresponding star ratings

Summary of Results and Conclusion

A fairly simple MLP Neural Network was able to comfortably beat the accuracy of Logistic Regression.The above results aligns to our intuition that since MLP's have the same objective function as the Logistic Regression, they should in principle perform better than the Logistic Regression, which they do.

2. Methodology

This entire project was executed according to the below guidelines:

  • Data set scraped from Amazon.com
    • Identification of relevant features (here reviews and star rating)
    • Pre - processing the reviews(stop words removal, stemming , lemmatizing etc.)
  • Determination of Optimal Machine Learning 2.0 Classification Algorithms (trying only MLPs here)
    • First apply the Logistic Regression Model to create our Baseline Accuracy
    • Now apply different combinations of MLP architectures to beat our baseline accuracy of Logistic Regression
  • Conclusion

Number_trials=1
def train_logistic(X, y, reg, scale=None):
    """
    Implements Logistic Regression
    
    Parameters:
    --------------
    X - Cleaned Data
    y - Target
    reg - Regularization Parameter (l1/l2)
    scale - Default is None. Specify scaler e.g StandardScaler or MinMaxScaler
    
    """
    C = [1e-8, 1e-4, 1e-3, 1e-2, 0.1, 0.2,0.4, 0.75, 1, 1.5, 3, 5, 10, 15,  20, 100, 300, 1000, 5000]

    score_train = []
    score_test = []
    weighted_coefs=[]
    print('Logistic - {}'.format(reg))
    for seed in tqdm(range(Number_trials)):
        training_accuracy = []  
        test_accuracy = []
        X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.25, random_state=seed)
        if scale:
            X_train = scale.fit_transform(X_train)
            X_test = scale.transform(X_test)
        for alpha_run in C:
            lr = LogisticRegression(C=alpha_run, penalty=reg).fit(X_train, y_train)
            training_accuracy.append(lr.score(X_train, y_train))
            test_accuracy.append(lr.score(X_test, y_test))
            coefs=lr.coef_ 
            weighted_coefs.append(coefs) #append all the computed coefficients per trial
                
        score_train.append(training_accuracy)
        score_test.append(test_accuracy)
    
    mean_coefs=np.mean(weighted_coefs, axis=0) #get the mean of the weighted coefficients over all the trials 
    #print(mean_coefs) 
    score = np.mean(score_test, axis=0)
    #return score.shape
    #coefs = lr.coef_
    #return C value, accuracy, column/feature name
#     top_predictor='NA'
    if (np.unique(y).size > 2):
        class_coeff={}
        C_max=C[np.argmax(score)]
        lr = LogisticRegression(C=C_max,penalty=reg).fit(X_train, y_train)
        coeff = lr.coef_
        class1 = lr.classes_
        for c,i in zip(class1,range(coeff.shape[0])):
#             print(coeff[i])
            class_coeff[c] = X.columns[np.argmax(coeff[i])]

    
    if (np.unique(y).size == 2):
        top_predictor=X.columns[np.argmax(np.abs(mean_coefs))]
        abs_mean_coefs = np.abs(mean_coefs[0,:])
        coefs_count = len(abs_mean_coefs)
        fig, ax = plt.subplots(figsize=(3,5))
        ax.barh(np.arange(coefs_count), sorted(abs_mean_coefs))
        #ax.barh(np.arange(coefs_count), mean_coefs[0, np.argsort(abs_mean_coefs)])
        ax.set_yticks(np.arange(coefs_count))
        ax.set_yticklabels(X.columns[np.argsort(abs_mean_coefs)])
        ax.set_title('Logistic Regression Feature Importance {0} regularized'.format(reg))
        #     print('Scaling used is: ', scale)    
        return ['Logistic ({0})'.format(reg), np.amax(score), \
                'C = {0}'.format(C[np.argmax(score)]), top_predictor]
    else:
        return ['Logistic ({0})'.format(reg), np.amax(score), \
                'C = {0}'.format(C[np.argmax(score)]), class_coeff]

def plot_history(history):
    acc = history.history['acc']
    val_acc = history.history['val_acc']
    loss = history.history['loss']
    val_loss = history.history['val_loss']
    x = range(1, len(acc) + 1)

    plt.figure(figsize=(12, 5))
    plt.subplot(1, 2, 1)
    plt.plot(x, acc, 'b', label='Training acc')
    plt.plot(x, val_acc, 'r', label='Test acc')
    plt.title('Training and Test accuracy')
    plt.legend()
    plt.subplot(1, 2, 2)
    plt.plot(x, loss, 'b', label='Training loss')
    plt.plot(x, val_loss, 'r', label='Test loss')
    plt.title('Training and Test loss')
    plt.xlabel('Number of epochs')
    
    plt.legend()

Discussion of Results

A fairly simple MLP Neural Network was able to comfortably beat the accuracy of Logistic Regression.The above results aligns to our intuition that since MLP's have the same objective function as the Logistic Regression, they should in principle perform better than the Logistic Regression, which they do. Hence we now have a Mental Model pertaining to text classification which would direct us towards Neural Networks rather than trying out ML1.0 ML models.

Summary and Conclusion

  • Among traditional models, Logistic Regression and SVMs are most suited for text classification From ML 1.0

  • Neural Networks (with sigmoid activations) are natural extensions of Logistic Regression. Hence are an obvious choice for handling sparse data vis--vis text classification

  • Relatively simpler neural networks can beat the most intricately conditioned Logistic Regression Models
